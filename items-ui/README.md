# Items UI - React Frontend

React frontend application for displaying and managing items with filtering, sorting, and pagination.

## Setup

1. Install dependencies (already done):
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will run on `http://localhost:5173`

## Features

- ✅ Fetch items from Express API
- ✅ Filter items by name
- ✅ Sort by rate, name, or creation date
- ✅ Pagination support
- ✅ Table layout for displaying items

## API Endpoint

The app connects to the backend API at `http://localhost:3000/items`

Make sure the backend server is running before starting the React app.
